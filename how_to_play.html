Basic Idea of the Game

To be the first player to place markers in a row either horizontally, vertically, diagonally or full cardboard, either one.

Each player has to select the number on their card, which will be marked as cross, only after they hear the number being called by the caller.

The player has to tally the number with the letter and number.

Game Play

To start the game, the Host distributes a Bingo card to each player.

The Host will start a new game to begin playing.

The caller will start calling random sequence of number and letter. The number will keep blinking for the next 10 seconds then move to the next number in the sequence.

Players then has to look for the letter & number on their cards. If the number appears on a player's card, select it to place a cross on it.

When a player has placed markers in a row, either horizontally, vertically, diagonally or full cardboard, the player has to call out "Bingo!"
